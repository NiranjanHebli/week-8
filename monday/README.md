# Day 44 (Monday) - Time Series Modeling

## Description

This task performs time series modeling on the Olist e-commerce dataset, which contains daily orders from February 2017 to August 2018. The task loads the data, cleans the data by handling missing values and outliers, visualizes the time series, decomposes the time series into trend, seasonality, and residuals, and models the residuals using SARIMA and Prophet.

## Key Learnings

- Time series modeling is a powerful tool for analyzing and forecasting sequential data.
- Handling missing values and outliers is an important preprocessing step in time series modeling.
- Decomposition of time series into trend, seasonality, and residuals can reveal underlying patterns in the data.
- SARIMA and Prophet are two popular libraries for modeling time series data.
