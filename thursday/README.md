# Day 47 (Thursday) - RNN-LSTM Stock Churn Analysis

## Description

This task implements a RNN-LSTM model to predict stock churn based on historical data. The task loads a dataset of historical stock prices, preprocesses the data, builds and trains a RNN-LSTM model, and evaluates its performance using mean absolute error and mean squared error. The task aims to predict the direction of stock churn (up or down) based on the historical data.

## Key Learnings

- RNN-LSTM models are a powerful tool for modeling sequential data and predicting future trends.
- The NumPy library provides an efficient and easy-to-use interface for building and training RNN-LSTM models.
- Preprocessing data is an important step in building an effective RNN-LSTM model.
- Evaluating the performance of a RNN-LSTM model is crucial in determining its effectiveness.
- The importance of hyperparameter tuning in RNN-LSTM models.
